# StockLens

> Automated Financial Reporting & Analytics System

A full-stack web application that pulls live stock data, generates AI-powered
insights, exports PDF reports, and schedules email summaries.

## Tech Stack
- **Frontend**: React, TypeScript, Vite, Recharts, Vercel
- **Backend**: Python, FastAPI, yfinance, Alpha Vantage, Railway
- **AI**: Google Gemini API
- **Data**: Yahoo Finance, Alpha Vantage, NSE/BSE, CSV/Excel upload
- **Infra**: PostgreSQL, Redis, Docker, GitHub Actions CI/CD

## Features
- Live stock dashboard with interactive charts
- Multi-ticker comparison
- AI-generated financial insights
- PDF report export
- Automated email report scheduler
- NSE/BSE Indian stock support
